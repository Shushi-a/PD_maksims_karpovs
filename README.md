# PD_maksims_karpovs

# Ievarijuma masu, nepiecišamo cukura masas un pagatavošanas preces kalkulators
  ##Apraksts
  Si programma kalkule:
  Cik daudz cukura (kg) ir niepieciesams lai pagatavot ievarijumu, ja ir zinams izmantoto abolu masa
  Cik daudz ievarijuma (kg) bus pagatavots rezultata pec recepta
  Cik daudz nauda (euro) sis ievarijuma recepts maksa  
  ##Palaisanas instrukcija
  Atvert failu iev_kalk.py, spiez F5 lai sakt programu, ievadit abolu masu (kg), abolu cena (eur/kg), cukura cena (eur/kg), iegut rezultatus, un velos labu laimi pagatavot sev ievarijumu!
